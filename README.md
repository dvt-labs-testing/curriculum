# devpost-curriculum

A Claude Code plugin marketplace from [Devpost](https://devpost.com).

## Plugins

### hackathon-in-a-plugin

A complete hackathon curriculum delivered as seven slash commands. Walk through spec-driven development from idea spark to working app in 3–4 hours.

**Commands:** `/scope` → `/prd` → `/spec` → `/checklist` → `/build` → `/iterate` → `/evaluate`

## Install

In Claude Code:

```
/plugin marketplace add devpost/devpost-curriculum
/plugin install hackathon-in-a-plugin@devpost-curriculum
```

## Requirements

- [Claude Code](https://claude.ai/code) v1.0.33+
